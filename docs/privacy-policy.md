# Personvernerklæring – SmartSparing

## Hvilke data vi samler inn

SmartSparing samler inn følgende data:

- Brukernavn
- Kryptert passord (passord-hash)
- Sparemål som brukeren selv oppretter
- Samtykkeinformasjon (tidspunkt og versjon)

Applikasjonen samler ikke inn personopplysninger som navn, e-postadresse eller fysisk adresse.

## Hvorfor vi samler inn disse dataene

Dataene er nødvendige for å:

- Kunne identifisere brukere i systemet
- La brukere opprette og administrere egne sparemål
- Sikre tilgangskontroll og beskyttelse av data
- Dokumentere brukerens samtykke

## Dataminimering

SmartSparing samler kun inn data som er strengt nødvendige for å levere tjenesten.

## Lagring og sikkerhet

Data lagres midlertidig i serverens minne.
Passord lagres aldri i klartekst, men som sikre kryptografiske hasher.

## Brukerens rettigheter

Brukere har rett til å:

- Få innsyn i egne data
- Slette kontoen sin og alle tilknyttede personopplysninger
- Trekke tilbake samtykke når som helst ved å slette kontoen

## Samtykke

Ved å opprette en konto gir brukeren aktivt samtykke til denne personvernerklæringen.
Samtykke kan når som helst trekkes tilbake ved sletting av konto.
